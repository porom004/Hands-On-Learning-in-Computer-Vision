## How the LinkedIn Post Generator Works

### Overview of the Process

Our LinkedIn post generator will take a blog post URL as input and, through a series of automated steps, generate a ready-to-share LinkedIn post. Here's a breakdown of the process:

1.  **Input: Blog Post URL:** The user provides the URL of a blog post.
2.  **Extraction of Content from the Blog:** The agent extracts the main content from the blog post, removing unnecessary elements like navigation menus and advertisements.
3.  **Summarization of the Blog Content:** The extracted content is summarized to create a concise and engaging summary suitable for a LinkedIn post.
4.  **Creation of a Professional LinkedIn Post:** Based on the summary, a professional LinkedIn post is crafted, including relevant hashtags and a link back to the original blog post.
5.  **Ready to Share Output:** The generated LinkedIn post is presented, ready for the user to review and publish.

### Importing Libraries

Now, let's import the required libraries into our Python script:

```python
import os
from dotenv import load_dotenv
from crewai import Agent, Task, Crew, Process
from bs4 import BeautifulSoup
import requests
```

### Creating a Helper Function

We'll create a helper function to fetch and extract the content from a blog post using `BeautifulSoup`:

```python
def fetch_blog_content(url):
    try:
        response = requests.get(url)
        response.raise_for_status()  # Raise an exception for bad status codes
        soup = BeautifulSoup(response.content, 'html.parser')
        # Adjust these selectors based on the target website's HTML structure
        article = soup.find('article') or soup.find('main')
        if article:
            return article.get_text(separator='\n', strip=True)
        else:
            return "Could not extract content from the provided URL."
    except requests.exceptions.RequestException as e:
        return f"Error fetching content: {e}"
```

### Defining Tools

Next, we define our tools. In this case, our primary tool will be the function to fetch blog content:

```python
from crewai.tools.tool import Tool

fetch_content_tool = Tool(
    name="Fetch Blog Content",
    description="Fetches and extracts the content from a given blog post URL.",
    func=fetch_blog_content,
    args_schema={
        "url": {
            "type": "string",
            "description": "The URL of the blog post.",
        }
    }
)
```

### LLM Configuration

For this example, we'll use Gemini models. You'll need to set up your API keys. Make sure your `GOOGLE_API_KEY` is set in your environment variables.

```python
load_dotenv()
os.environ["GOOGLE_API_KEY"] = os.getenv("GOOGLE_API_KEY")
```

### Agent Creation

Now, let's create the agents. We'll have two agents: one to extract and summarize content and another to write the LinkedIn post.

```python
from langchain_google_genai import ChatGoogleGenerativeAI

# Configure LLMs
llm_extraction = ChatGoogleGenerativeAI(model="gemini-1.5-pro", temperature=0.1)
llm_writing = ChatGoogleGenerativeAI(model="gemini-1.5-pro", temperature=0.2)

# Content Extractor Agent
content_extractor_agent = Agent(
    role='Content Extractor',
    goal='Extract and summarize content from blog posts.',
    backstory='I am an expert at extracting the core content from any given blog post and preparing it for summarization.',
    tools=[fetch_content_tool],
    llm=llm_extraction,
    verbose=True
)

# LinkedIn Post Writer Agent
linkedin_post_writer_agent = Agent(
    role='LinkedIn Post Writer',
    goal='Write engaging and professional LinkedIn posts based on extracted content.',
    backstory='I am a social media expert skilled in crafting compelling LinkedIn posts that capture attention and drive engagement.',
    llm=llm_writing,
    verbose=True
)
```

### Task Definition

Define the tasks for each agent:

```python
# Extract and Summarize Task
extract_and_summarize_task = Task(
    description='Extract the content from the blog post at {url} and create a concise summary.',
    agent=content_extractor_agent,
    input={"url": "https://www.example.com/blog-post"}, # Replace with your blog post URL
    expected_output='A concise summary of the blog post content.',
)

# LinkedIn Post Writing Task
linkedin_post_writing_task = Task(
    description='Write a professional and engaging LinkedIn post based on the following summary: {summary}. Include relevant hashtags and a link to the original blog post. The post should be under 200 words and have a friendly tone.',
    agent=linkedin_post_writer_agent,
    expected_output='A ready-to-share LinkedIn post.',
)
```

### Pre-assembly: Creating the Pipeline (Sequential Process)

Finally, create the crew and define the tasks.

```python
# Assemble the Crew
crew = Crew(
    agents=[content_extractor_agent, linkedin_post_writer_agent],
    tasks=[extract_and_summarize_task, linkedin_post_writing_task],
    process=Process.sequential,
    verbose=2, # Set verbose to 2 for detailed execution logs
)
```

## Execution and Results

### Providing a Blog URL as Input

Replace `"https://www.example.com/blog-post"` in the `extract_and_summarize_task` with the actual URL of the blog post you want to use.

### Running the Code

Run the Python script. The agents will work sequentially, extracting the content, summarizing it, and generating the LinkedIn post.

```python
# Run the crew to generate the LinkedIn Post
result = crew.kickoff()
print("\nGenerated LinkedIn Post:\n", result)
```
